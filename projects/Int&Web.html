Internet

A worldwide network that connects computers so they can share information.

Web

System of Websites and pages that you use on the Internet

How are the Internet and Web different?

The Internet is the network itself. The Web is the information (websites) on that network.

ASCII

A code that uses numbers to represent English letters and symbols.

Unicode

A bigger code that can show characters from almost every language.

UTF-8

A common way to save Unicode so computers can read it easily.

URL

Rules for sending and recieving web pages

Domain Name

The name of the website

HTTP

Rules for sending and recieving web pages

HTTPS

The Secure version of HTTP that keeps the data provate

IP Address

A unique number that identifies each computer or device online.

DNS

Like a phone book that changes website names into IP addresses.

W3C Recommendations

Official guidelines that explain how websites should be built so they work everywhere.

XML

A way to store and organize data with custom tags

XHTML

A stricter version of HTML that follows XML rules

HTML

Basic language used to build web pages

HTML 5

THe newest version of HTML, the extra features like video and audio support
