Engagement Questions

1) Why do we need a viewport meta tag?

It makes the page fit the screen size on phones. Without it, the page looks tiny and zoomed out.

2) If I set flex-direction: column and later set flex-direction: row, which one works?

The one that comes later in the CSS. If two rules affect the same thing, the last one wins unless another rule is stronger.

3) How can you change CSS for a phone?

Use a media query, like:

@media only screen and (max-width: 600px){
          /* phone styles */
        }
